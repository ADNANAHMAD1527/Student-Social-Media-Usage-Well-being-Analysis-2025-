# 📊 Student Social Media Usage & Well-being Analysis (2025)

## 📌 Project Overview

This project analyzes students' social media usage patterns using **Microsoft Excel** to understand their impact on academic performance, sleep quality, mental health, relationship conflicts, and addiction levels.

An interactive dashboard was built using **Pivot Tables, Pivot Charts, KPIs, and Slicers** to transform raw survey data into meaningful insights and recommendations for promoting healthier digital habits.

---

## 🎯 Business Problem

Educational institutions and parents often lack clear insights into students' social media behavior and its association with academic performance and well-being. This project provides a data-driven analysis to identify usage trends and support informed decision-making.

---

## 🎯 Objectives

- Analyze students' social media usage patterns.
- Identify the most popular social media platforms.
- Analyze academic performance impact.
- Evaluate sleep patterns and mental health.
- Compare usage across gender and academic levels.
- Generate actionable insights and recommendations.

## 🔍 Key Insights

- 705 students participated with an almost equal gender distribution.
- Undergraduate students formed the largest student group.
- Students spend an average of **4.92 hours/day** on social media.
- Instagram is the most popular social media platform.
- WhatsApp users recorded the highest average daily usage.
- Students reporting academic impact also reported lower average sleep.
- Average sleep duration was **6.66 hours/day**.
- Average mental health score was **6.23/10**.
- India contributed the highest number of survey responses.
